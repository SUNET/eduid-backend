Introduction

Group database functionality.

Install
-------

For development and testing,
make a virtualenv, activate it, and install like::

  $ virtualenv venv
  $ source venv/bin/activate
  $ pip install -r test_requirements.txt

Test
----

Once installed, test it with::

  $ source venv/bin/activate
  $ pytest
